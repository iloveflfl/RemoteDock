A personal dock for
Raspberry Pis,
servers, and SSH devices

RemoteDock는 Windows에서 SSH 장비를 등록하고, 상태를 감지하고, SSHFS-Win으로 드라이브를 마운트하고, 터미널·VS Code·웹·명령 프리셋·백업까지 연결하는 개인 서버 도킹 앱입니다.

RemoteDock lets you register SSH-capable devices, monitor their status, mount them with SSHFS-Win, and jump into Explorer, Terminal, VS Code, web URLs, command presets, and backups.

핵심 기능

Highlights

파일 마운트부터 서버 작업까지 한 화면에서.

From drive mounting to server operations in one place.

장비 등록과 자동 감지

Device registry & auto detect

SSH 장비, PC, 노트북, VPS를 등록하고 온라인 상태를 주기적으로 확인합니다.

Register SSH devices, PCs, laptops, and VPS machines, then watch their online status.

SSHFS-Win 마운트

SSHFS-Win mounting

WinFsp + SSHFS-Win 기반으로 원격 경로를 Windows 드라이브처럼 붙입니다.

Mount remote paths as Windows drives through WinFsp and SSHFS-Win.

시각적 상태 대시보드

Visual status dashboard

온라인, 마운트, CPU, RAM, 디스크 상태를 간단한 바와 색상으로 표시합니다.

Show online, mount, CPU, RAM, and disk state with compact bars and status colors.

터미널·VS Code·웹 연결

Terminal, VS Code, web

선택한 장비를 터미널, VS Code Remote SSH, 웹 URL로 바로 엽니다.

Jump into Terminal, VS Code Remote SSH, or web URLs for the selected device.

명령 프리셋과 즐겨찾기

Command presets & favorites

자주 쓰는 SSH 명령과 원격 폴더를 버튼처럼 관리합니다.

Keep frequent SSH commands and remote folders ready as quick actions.

키 사본과 권한 보정

Key copy & ACL hardening

등록한 SSH key를 앱 데이터 폴더에 복사하고 Windows OpenSSH 권한 오류를 줄입니다.

Copy selected SSH keys into the app data folder and harden ACLs for Windows OpenSSH.
